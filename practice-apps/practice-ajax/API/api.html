get /pets/

Read the list of pets

  • Request
  • Response

Headers

  • token: (string)

    Authentication token that the server provided. - only if you're running the server in the "auth" mode

Query Parameters

  • name: (string)

    A pet's name - used to get only pets whose names start with this string

  • pageNo: (string)

    Page number - starting at 0

  • pageSize: (string)

    Page size - default

HTTP status code 200

The entire list of pets or the filtered/paginated list if query params were passed.

Body

Media type: application/json

Type: object

Example:

[ {
    "name": "Joy",
    "type": "cat",
    "id": 151398,
    "created": 1533823910803
},{
    "name": "Bob",
    "type": "dog",
    "id": 189796,
    "created": 1533822727365
}]

HTTP status code 401

Not authenticated - only if you're running the server in the "auth" mode
